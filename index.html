Random Forest Visualization App

1. Indented Aggregate Tree

This visualization is built with d3, javascript, and python. It displays aggregated feature variables at node positions based on the decision trees generated by a random forest classifier.

See the article and visualization.

The repository contains the code and a useful overview of the project.

2. Random Forest and Ensemble Filtering

This is an application that fits a random forest classifier based on a data set, and filters the ensemble of the decision trees generated by the random forest by quantifying the trees by hamann similarity measure (more information is provided in the link below). The application is implemented with RShiny.

See the live demo with more information here.

M.Sc. Project

This project involves the development of tree-based supervised methods to discern different waveform modulation types from multiple short range devices. The tree-based methods include classification tree, random forest, and variations of the two. These variations include feature space partitioning and feature selection.

The repository contains a tutorial to run the simulations and experiments.

The paper will be posted here in the near future when it is published.

Social Network Analysis

This report analyzes a network of friends in facebook. It further compares against a random graph and several networks from the stanford database.

The repository contains the code.

Stochastic Gradient Boosting with Squared Error Epsilon Loss

This paper describes the implementation of a robust loss function for gradient boosting. The implementation is in R, and the repository contains the code for the project.
